Unlike a pure JavaScript project, which takes no time to get started, GWT, like any other Java projects, requires some setting up to do to get started. For this tutorial, we will use Maven to manage the build and artifacts. The project will follow Maven project structure.

Shown all the way at the bottom of the directory structure, you will find a file named pom.xml. This is a Maven project descriptor. You will find more information about in the next chapter, Project Object Model.
